# Bank-Kata_C-Sharp
Pair programming session of the Test Driven Development (TDD) Bank Kata in C#, with an Outside-In approach.

Using:
* [xUnit](https://xunit.net/) as the testing framework 
* [Moq4](https://github.com/moq/moq4) as the mocking one.
